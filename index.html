Как добавить новый репозиторий:
1. Когда размер достигнет ~800 МБ, создайте новый репозиторий:
  github.com/new
  Название: pony_storage_2 (Public)
2. Зайдите в Cloudflare Worker → gallery-trigger → Edit code
3. Найдите строку const GITHUB_REPO и замените на:
  const GITHUB_REPO = 'pony_storage_2';
4. Нажмите Save and Deploy
